Aplikasi Keuangan Kasir Berbasis Mobile – UD Jaya

Latar Belakang

Pencatatan transaksi keuangan di UD Jaya masih dilakukan secara manual sehingga kurang efisien dan rawan kesalahan. Aplikasi kasir berbasis mobile dikembangkan untuk membantu pencatatan dan pengelolaan keuangan yang lebih terstruktur.

Tujuan

Menyediakan aplikasi kasir mobile untuk mencatat transaksi dan menyajikan laporan keuangan harian secara cepat dan akurat.

Peran

UI/UX Designer & Mobile App Developer

Tools

Figma
Firebase
VS Code
Cloudinary

Hasil

Aplikasi kasir mobile berhasil dikembangkan dan meningkatkan efisiensi pencatatan keuangan hingga ±30%.
