Aplikasi Gudang “E-Stock”

Latar Belakang

Pengelolaan stok gudang masih dilakukan secara manual sehingga menyulitkan pencatatan dan pemantauan barang. Aplikasi E-Stock dikembangkan untuk membantu pengelolaan stok agar lebih terstruktur dan akurat.

Tujuan

Menyediakan aplikasi gudang untuk pencatatan stok masuk dan keluar secara real-time.

Peran

UI/UX Designer & Mobile App Developer.

Tools

Figma
VS Code
MySQL

Hasil

Redesign front-end menghasilkan tampilan yang lebih modern dan mudah digunakan, serta meningkatkan efisiensi penggunaan aplikasi hingga ±20%.
